  1) Identifiers.
     - Any names called Identifiers such as variable , method and class name.
     Roles:

     2) ca$h - invalid
     3) total123 - valid
     4) 123total - invalid
     5) all@vars - invalid
     6) java2share - valid
     7) if - reserved words are invalid to use
     8) __add__ : valid


     _x   : Called protected variables
     __x  : private variables
     __x__: Magic variables
